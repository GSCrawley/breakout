Enhanced Breakout

A modernized version of the classic Breakout game with multiple levels, power-ups, and 3D graphics.

How to Play:

Use the left/right arrow keys or mouse to move the paddle. Destroy all bricks to advance to the next level.

Power-ups:

Widen Paddle
Narrow Paddle
Slow Ball
Fast Ball
Multi-ball
Extra Life

Press P to pause the game at any time.
